Как откатить коммит на github

Опубликовано 2018-01-11

Иногда бывают ситуации, когда надо удалить (откатить) коммит обратно, для этого:
1. Получаем хэш-код коммита, к которому хотим вернуться.
2. Заходим в папку репозитория (локально) и пишем в консоль:

$ git reset --hard <хэш-код коммита> 
$ git push --force

Вот и вся операция.
