Генерация сайта на HUGO

 Опубликовано 28 May, 2017
За прошедший год идей для статей было много, но ни одна из них не превратился в статью. За это время была переустановлена система и все исходники блога были удалены, осталось только сгенерированная статика. Чтоб не забыть, небольшие заметки: 1) Установить Hugo. Для этого воспользуемся Homebrew. Пишем следующую команду в терминале: /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" После успешной инсталляции приступаем к установке Hugo, для этого в терминале пишем: brew install hugo Hugo установлен.
