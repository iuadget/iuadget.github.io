Смена пароля в MySQL сервере

Jul 2017
mysql macos
1 мин на чтение

1. Смена пароля

Поставил на macOS Sierra MySQL сервер. При установке MySQL выдал временный пароль:

В новой версии MySQL временные пароль стали иметь срок действия. При установке MySQL генерирует временный пароль, но срок его действия сразу кончается. И при первом подключении к MySQL выдается следующая ошибка:

ERROR 1862 (HY000): Your password has expired. To log in you must change it using a client that supports expired passwords.

Для изменения пароля необходимо в терминале ввести:

/usr/local/mysql/bin/mysqladmin -u root -p'temppassword' password 'newpassword'

2. Удаление MySQL

Для удаления MySQL сервера необходимо:

1) остановить MySQL сервер;

2) в терминале ввести следующие команды:

sudo rm /usr/local/mysql
sudo rm -rf /usr/local/mysql*
sudo rm -rf /Library/StartupItems/MySQLCOM
sudo rm -rf /Library/PreferencePanes/My*
sudo rm -rf /Library/Receipts/mysql*
sudo rm -rf /Library/Receipts/MySQL*
sudo rm -rf /private/var/db/receipts/*mysql*
