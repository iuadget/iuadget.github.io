Баги iOS11 beta

Опубликовано 2017-07-18

За два дня работы на iOS11 beta было найдено пару багов:

1. Анимация при выполнения стандартных действий:

2. Невозможно переместить приложение из папки

3. Артефакты анимации приложений при изменении/переименовании папки

4. Локализация в AppStore

5. Перекрытие кнопкой иконки приложения


Все найденные баги были отправлены в Apple.
